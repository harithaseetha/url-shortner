# URL Shortener Demo

Minimal FastAPI-based URL shortener.

Requirements
- Python 3.10+

Install

```bash
python -m pip install -r requirements.txt
```

Run (development)

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

Run tests

```bash
pytest -q
```

Verify with curl

```bash
curl -i -X POST http://127.0.0.1:8000/shorten \
  -H "Content-Type: application/json" \
  -d '{"target_url": "https://example.com"}'

curl -i http://127.0.0.1:8000/<alias>
curl -i http://127.0.0.1:8000/<alias>/meta
```

Notes
- Uses SQLite by default and creates the local DB file only when the app runs.
- Short alias generated by base62 encoding of DB `id` when `custom_alias` is not provided.
- Custom aliases enforce uniqueness; collisions return HTTP 409.
